# MémoCuisto

Projet réalisé dans le cadre du module **PS6** suivi en SI3 à Polytech Nice Sophia.

## Description

MémoCuisto est un jeu de stimulation cognitive inspiré d'une activité du quotidien : préparer une recette.  
Le joueur doit suivre plusieurs étapes simples qui sollicitent différentes fonctions cognitives comme la mémoire, l'organisation et la planification.

Le jeu est conçu pour être **accessible et progressif**, notamment pour des personnes ayant des troubles cognitifs légers à modérés (maladie d'Alzheimer).

## Principe du jeu

Le jeu se déroule en **trois étapes successives** :

1. **Mémoriser et retrouver les ingrédients**  
   Le joueur observe une liste d'ingrédients pendant quelques secondes puis doit retrouver ces éléments dans un rayon de supermarché.

2. **Ranger les courses**  
   Les ingrédients doivent être placés au bon endroit :  
   - réfrigérateur  
   - placard

3. **Remettre les étapes de la recette dans l'ordre**  
   Le joueur organise les étapes de la recette en utilisant un système de drag&drop.

## Système d'aides

Un **timer discret** fonctionne en arrière-plan.  
Si le joueur reste bloqué trop longtemps, des aides apparaissent progressivement :

- **Aide simple** : indication légère (numéro, symbole)
- **Aide moyenne** : un élément correct est suggéré
- **Aide forte** : plusieurs éléments sont placés automatiquement
- **Réponse entière** : la solution est affichée

Ce système permet d'éviter la mise en échec et de maintenir la progression du joueur.

## Fonctionnalités principales

- Interface simple et lisible
- Système d'aides progressives
- Timer optionnel affichable
- Drag&Drop pour certaines étapes
- Profil aidant avec statistiques et notes
- Choix de thème visuel
