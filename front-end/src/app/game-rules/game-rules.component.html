Le but du jeu 👨‍🍳

Bienvenue dans MemoCuisto ! Ce jeu vous accompagne dans une activité du quotidien : faire les courses, ranger les achats, puis réaliser une recette simple. L’objectif est de stimuler la mémoire, l’organisation et la concentration tout en s’amusant, à votre rythme.

Comment jouer ? 🧩

Choix de la recette

Après avoir choisi votre recette pour la partie, retenez bien les ingrédients avant de passer à la suite.

1. La liste de courses 🛒

Essayez de retrouvez-les ingrédients de votre recette dans le supermarché. (PS: cliquez sur le panier en haut à droite si vous avez besoin d'un rappel !)

2. Le rangement 🧺

Une fois les ingrédients récupérés, rangez-les au bon endroit dans la cuisine. Vous pouvez les ranger dans le placard, le frigo ou le congélateur.

3. La recette 🍰

Replacez les étapes de la recette dans le bon ordre pour la réaliser correctement.

Les modes de jeu 🎮

Mode Autonomie

Dans ce mode de jeu, vous jouez seuls les 3 étapes précédentes (courses, rangement, étapes de la recette), les indices peuvent être activés.

Mode Duo

Dans ce mode de jeu, une personne choisit ou créer une recette et l'objectif est de faire deviner cette recette à l'autre. Vous jouez avec un proche les étapes 1 et 2 mentionnées précédemment (courses, rangement), ainsi qu'une troisième étape dans laquelle un des joueurs devra deviner le nom de la recette en cours. Pas d'indices dans ce mode de jeu.

Niveaux de difficulté ⚙️

Il existe 3 niveaux de difficulté qui font varier le nombre d'ingrédients dans la recette.

Aides et encouragements 💬

Les aides et encouragements peuvent être activités dans les configurations par le soignant. Elles comprennent par exemple :

  • des indices modifiable selon le besoin
  • des messages d'encouragements
  • des rappels après une longue période d'inactivité

Ici, pas de stress : le plus important est de participer et de progresser à votre rythme !
